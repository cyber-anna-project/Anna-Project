# 🤖 Anna AI Project

Anna AI adalah asisten virtual berbasis **Python** yang dikembangkan menggunakan framework **Kivy**. Proyek ini dirancang untuk berjalan di lingkungan mobile (Android) melalui **Termux**.

## 🚀 Fitur Utama
- **Asisten Personal:** Interaksi berbasis teks dan perintah.
- **Mobile Optimized:** Dibangun khusus untuk efisiensi di perangkat Android.
- **Security Oriented:** Pengembangan fokus pada praktik keamanan kode dan manajemen data.

## 🛠️ Teknologi yang Digunakan
- **Bahasa:** [Python 3.x](https://www.python.org/)
- **Framework:** [Kivy](https://kivy.org/)
- **Environment:** [Termux](https://termux.dev/) & [Buildozer](https://github.com/kivy/buildozer)
- **Version Control:** Git & GitHub

## 🛡️ Catatan Keamanan
Proyek ini mengikuti standar keamanan dasar:
- Penggunaan `.gitignore` untuk melindungi data sensitif.
- Manajemen kredensial menggunakan Personal Access Token (PAT).
- Enkripsi database (Tahap Pengembangan).
